# Shubham Srivastava — Personal Portfolio

A professional personal portfolio built as a first-year B.Tech Computer Science Engineering mini project at Woxsen University.

## Technology

- HTML5 for semantic page structure
- CSS3 for layout, visual design, and responsive behaviour
- Vanilla JavaScript for interactive behaviour

No frameworks, libraries, or UI kits are used.

## Project structure

```text
portfolio/
├── index.html          # Semantic page content
├── style.css           # Visual presentation (Phase 3 onward)
├── script.js           # Interactive behaviour (Phase 5 onward)
├── assets/
│   ├── images/         # Portfolio images and project screenshots
│   └── icons/          # Any reusable local icon assets
└── README.md           # Project documentation
```

## Development roadmap

1. Plan the structure and build the Navbar + Hero HTML.
2. Complete the remaining semantic HTML sections.
3. Build the responsive CSS layout and visual system.
4. Add refined CSS animations.
5. Add JavaScript interactions.
6. Optimise performance and accessibility.
7. Deploy the portfolio.
8. Prepare the project presentation and viva.
